# Twitter Flags

A browser extension that displays country flags on Twitter/X profile hover cards. Works on Chrome and Firefox. 

![review](https://github.com/user-attachments/assets/da4fce1a-c6c8-475a-91bd-0f2049e2e6cc)

## Installation

### Chrome (Developer Mode)

1. Download this repository (Code -> Download ZIP) and extract it (or clone the repository)
2. Open Chrome and go to `chrome://extensions/`
3. Enable **Developer mode** (toggle in top right)
4. Click **Load unpacked**
5. Select the `twitter-flags` folder
6. Visit Twitter/X and hover over any username to see their country flag

## Installation

1. Download this repository (Code → Download ZIP) and extract it (or clone the repository)

### Chrome (Developer Mode)

2. Open Chrome and go to chrome://extensions/
3. Enable Developer mode (toggle in top right)
4. Click "Load unpacked"
5. Select the twitter-flags folder

### Firefox (Temporary Add-on)

2. Open Firefox and go to about:debugging#/runtime/this-firefox
3. Click "Load Temporary Add-on…"
4. Select the manifest.json file inside the twitter-flags folder
