# Twitter Flags

A Chrome extension that displays country flags on Twitter/X profile hover cards.

![review](https://github.com/user-attachments/assets/da4fce1a-c6c8-475a-91bd-0f2049e2e6cc)

## Installation

### Chrome (Developer Mode)

1. Download or clone this repository
2. Open Chrome and go to `chrome://extensions/`
3. Enable **Developer mode** (toggle in top right)
4. Click **Load unpacked**
5. Select the `twitter-flags` folder
6. Visit Twitter/X and hover over any username to see their country flag
